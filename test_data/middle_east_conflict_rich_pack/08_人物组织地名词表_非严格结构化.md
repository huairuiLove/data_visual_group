# 人物、组织与地名词表：非严格结构化版本

## 说明

本词表故意不是严格数据库表，而是模拟记者、研究员或分析员手记。它适合测试实体识别、别名合并、组织类型判断和跨文本链接。

## 国家与准国家实体

**以色列 / Israel**：1948 年建国。核心安全议题包括周边国家战争、巴勒斯坦武装、真主党、伊朗导弹与核项目、红海航运威胁。政府、军方、情报机构和定居者运动在不同事件中扮演不同角色。

**巴勒斯坦 / Palestine**：既指历史地理，也指巴勒斯坦民族政治实体。外交上常由 PLO/PA 代表；加沙由哈马斯控制多年；西岸则由 PA 部分治理但以色列保留大量安全控制。

**黎巴嫩 / Lebanon**：主权国家，但国家军队不完全垄断武力。真主党是重要军事和政治力量。南黎巴嫩、利塔尼河、贝鲁特南郊、提尔、纳巴提耶常出现在报道中。

**伊朗 / Iran**：1979 年革命后成为反美、反以色列区域力量。通过导弹、无人机、革命卫队和盟友网络影响黎巴嫩、叙利亚、伊拉克、也门和巴勒斯坦战线。

**叙利亚 / Syria**：2011 年内战，2024 年阿萨德倒台，2026 年处于过渡与重建阶段。仍有 ISIS、库尔德问题、土耳其影响、化武清除和制裁调整等议题。

**伊拉克 / Iraq**：2003 年美国入侵后进入长期重建和教派政治。什叶派民兵、库尔德自治、ISIS 残余、美军存在和伊朗影响是关键词。

**也门 / Yemen**：胡塞控制萨那及北部大片地区，国际承认政府和南方力量控制其他地区。长期人道危机与红海航运威胁叠加。

## 组织与武装

**哈马斯 / Hamas**：巴勒斯坦伊斯兰主义组织，控制加沙多年。既是武装组织，也有治理机构。2023 年 10 月 7 日袭击以色列南部后成为以色列主要军事目标。

**巴勒斯坦伊斯兰圣战 / Palestinian Islamic Jihad**：较小但军事活跃的巴勒斯坦武装组织，与伊朗关系常被报道。

**PLO / 巴勒斯坦解放组织**：历史上代表巴勒斯坦民族运动，1990 年代与以色列签署奥斯陆协议。

**PA / 巴勒斯坦权力机构**：奥斯陆后建立，治理西岸部分地区。面临合法性危机、财政压力和与以色列安全合作争议。

**真主党 / Hezbollah**：黎巴嫩什叶派武装和政党，受伊朗支持。其身份包括军事组织、政治党派、社会服务网络和区域代理节点。2006 年与以色列战争，2023-2026 年与以色列边境战升级。

**胡塞 / Houthis / Ansar Allah**：也门北部运动，控制萨那等地。与伊朗关系密切但有本土议程。2023 年后通过红海袭船将也门内战与以巴/伊朗-以色列冲突连接。

**伊斯兰革命卫队 / IRGC**：伊朗关键军事政治机构，圣城旅负责外部行动和盟友网络。常与导弹、无人机、叙利亚、伊拉克民兵、真主党和胡塞相关。

**ISIS / Islamic State**：2014 年在伊拉克和叙利亚占领大片领土，后失去“哈里发国”领土，但地下网络仍在叙利亚和伊拉克活动。

**UNRWA**：联合国近东巴勒斯坦难民救济和工程处，为巴勒斯坦难民提供教育、医疗和社会服务。战争中经常成为援助、争议和政治攻击焦点。

**OCHA**：联合国人道事务协调厅，发布人道更新、伤亡、位移、援助准入、基础设施和保护风险数据。

**WFP**：世界粮食计划署，关注粮食安全、援助分发和饥饿风险。

**UNHCR**：联合国难民署，关注难民、庇护、返还和境内流离失所者保护。

## 地点和通道

**加沙地带 / Gaza Strip**：狭小、人口密集、长期封锁和战争。地名包括加沙城、汗尤尼斯、拉法、贾巴利亚、努赛赖特、代尔拜拉赫。

**约旦河西岸 / West Bank**：包括 A/B/C 区、定居点、检查站、杰宁、纳布卢斯、希伯伦、图勒凯尔姆、拉马拉等。

**东耶路撒冷 / East Jerusalem**：1967 年被以色列占领并随后吞并；国际地位争议极大。

**戈兰高地 / Golan Heights**：1967 年被以色列占领，后被以色列吞并，国际承认争议。

**南黎巴嫩 / Southern Lebanon**：真主党与以色列边境战核心区域。利塔尼河以南经常是停火和撤军条件焦点。

**贝鲁特南郊 / Dahiyeh**：真主党重要政治和社会基础区，常成为以色列打击或威胁目标。

**霍尔木兹海峡 / Strait of Hormuz**：全球油气运输 chokepoint，伊朗威慑和制裁谈判核心。

**曼德海峡 / Bab al-Mandab**：红海南端连接印度洋，胡塞袭船使其成为全球航运风险节点。

**苏伊士运河 / Suez Canal**：欧洲-亚洲航运主通道，红海风险会使船只绕行好望角，增加成本。

## 人物

**本雅明·内塔尼亚胡 / Benjamin Netanyahu**：以色列总理，长期主导以色列右翼安全政策，2023 年后战争决策核心人物。

**Ahmed al-Sharaa**：叙利亚阿萨德倒台后过渡时期领导人，Reuters 2026 年报道其推动内阁改组、化武清除和反 ISIS 合作相关议题。[Reuters_Syria_Cabinet_2026_05_09]

**Bashar al-Assad / 巴沙尔·阿萨德**：叙利亚前总统，2011 年后镇压抗议并引发内战，2024 年倒台。

**伊朗最高领袖与革命卫队领导层**：具体人物随时期变化，但在核政策、导弹、区域盟友和对以色列策略中居核心位置。

**美国总统与国务卿**：在 2026 年 Reuters 报道中，美国政府推动伊朗、以色列、黎巴嫩相关谈判，并将霍尔木兹、制裁和停火联系起来。[Reuters_IranIsrael_2026_06_08]

## 概念词

**停火 / ceasefire**：不一定等于和平。可能是局部、临时、有条件、未执行或被各方互相指控违反。

**代理人战争 / proxy war**：不能简单理解为完全控制。代理人可能有自主议程，也可能在某些议题上与赞助国不一致。

**占领 / occupation**：在巴以语境中涉及国际法、军事控制、定居点和主权争议。

**封锁 / blockade/siege**：可能指陆地口岸、海域、空域、燃料、货物、金融和人员流动限制。

**人道准入 / humanitarian access**：包括援助卡车、通道许可、安全保障、检查机制、燃料、仓储、分发和地方秩序。
