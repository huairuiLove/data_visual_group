# 使用说明：为什么这组文件适合测试知识图谱生成

这组文件不是一份教科书式单线叙述，而是一个“中东冲突语料包”。它混合了长历史、短新闻、跨国关系、人道数据、外交谈判、军事术语、组织名称、地名变体、同一事件的不同说法，以及不完全一致的数据口径。这样的设计更接近真实世界新闻和历史材料，适合测试知识图谱系统在以下任务中的表现：

1. 从长文本中抽取“事件-时间-地点-参与方-结果-数据”的关系。
2. 区分国家、非国家武装、国际组织、临时政府、难民机构和媒体来源。
3. 将不同文件中的同一实体合并，例如“真主党/Hezbollah/黎巴嫩真主党”、“胡塞/Houthis/安萨尔 Allah”、“加沙地带/Gaza Strip”。
4. 识别同一冲突的不同层级：本地战斗、代理人战争、区域航运危机、全球能源价格冲击。
5. 处理“时间跨度大”与“新闻动态新”的矛盾：1917 年的贝尔福宣言、1948 年战争、1967 年占领、1993 年奥斯陆进程、2003 年伊拉克战争、2011 年阿拉伯之春、2023 年加沙战争、2026 年伊朗-以色列与红海局势。

文件分工：

- `01_历史长线_从帝国秩序到民族国家.md`：长时段背景，强调帝国崩解、殖民委任统治、民族主义和边界问题。
- `02_巴以与阿以战争_一条主轴.md`：从 1917 到 2026 的巴勒斯坦/以色列问题叙述。
- `03_新闻报道包_2023_2026区域化战争.md`：模拟新闻综述体，聚焦 2023-2026 的事件链。
- `04_加沙与西岸_人道危机和治理碎片化.md`：人道、治理、人口位移、援助准入。
- `05_黎巴嫩与真主党_边境战和国家脆弱性.md`：黎巴嫩线，含 1982、2006、2023-2026。
- `06_伊朗_红海_能源通道与代理人网络.md`：伊朗、胡塞、霍尔木兹、曼德海峡、航运保险。
- `07_叙利亚_伊拉克_也门_三条长期内战线.md`：叙利亚、伊拉克、也门如何成为区域竞争平台。
- `08_人物组织地名词表_非严格结构化.md`：半结构化词条，用于实体抽取。
- `09_数据与口径说明.md`：收集关键数字、来源差异和口径陷阱。
- `10_来源书目与可继续抓取方向.md`：来源、API/数据集建议和爬取思路。

注意：本资料包故意保留一些复杂性，例如同一事件可能同时被描述为“战争”“升级”“军事行动”“报复性打击”“边境冲突”；同一组织既可能是军事行为体，也是政党、社会服务网络或治理力量。这些歧义正是知识图谱生成时最常见的问题。
